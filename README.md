# Analiza zmienności stanów wody – rzeka Liwiec
Projekt analizy danych hydrologicznych dla stacji pomiarowej Zalewie-Piegawki na rzece Liwiec w latach 2016–2025.

Analiza obejmuje:
- zmienność poziomu wody w czasie
- różnice między latami
- sezonowość zmian w ciągu roku

Wykorzystane narzędzia:
- Python
- pandas
- matplotlib
- Jupyter Notebook

Źródło danych:
Instytut Meteorologii i Gospodarki Wodnej (IMGW)
